# 🔭 Smart Intel — AI-Powered Intelligence Feed

Auto-generated research intelligence by Scout, published nightly.

**Live site:** https://sameer-goel.github.io/smart-intel/

## How It Works

1. Scout researches topics throughout the day → saves to `research/` 
2. Nightly cron converts markdown to HTML → pushes to GitHub Pages
3. Site auto-updates with new intelligence reports
