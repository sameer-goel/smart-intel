# Newsletter System — Smart Intel Weekly

## Overview
Weekly AI-generated intelligence newsletter covering AI, Cloud, Security, Geopolitics, Business, and Technology.

**Tagline:** "AI reads everything. You get the signal."  
**Frequency:** Every Monday, 8 AM CET  
**Format:** 18 insights (top 3 per category) with source links

## Architecture

```
insights.json (updated nightly by publish.js)
        ↓
newsletter.js --generate → HTML email
        ↓
Buttondown API → Subscribers
```

## Setup Steps

### 1. Create Buttondown Account
1. Go to https://buttondown.com/register
2. Sign up with `agenticinbox33@gmail.com`
3. Set newsletter username to `smartintel`
4. Get API key from Settings → API

### 2. Configure API Key
```bash
# Add to environment
export BUTTONDOWN_API_KEY="your-api-key-here"
```

### 3. Update Form URLs
Replace placeholder URLs in:
- `docs/index.html` — form action
- `docs/subscribe.html` — form action

### 4. Send Newsletter
```bash
# Generate HTML email
node scripts/newsletter.js --generate > /tmp/newsletter.html

# Send via Buttondown API
curl -X POST https://api.buttondown.com/v1/emails \
  -H "Authorization: Token $BUTTONDOWN_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "subject": "🔭 Smart Intel Weekly — Mar 17, 2026",
    "body": "<html content here>",
    "status": "draft"
  }'
```

### 5. Automate (Cron)
Add to OpenClaw cron: Every Monday 8 AM CET
- Generate newsletter from latest insights.json
- POST to Buttondown API as draft (or auto-send)
