🔬 TECHNOLOGY

What Is OpenClaw?

An open-source autonomous AI agent OS created by Peter Steinberger (Austrian, ex-PSPDFKit founder). Started as "Clawdbot" (Nov 2025), renamed to OpenClaw (Jan 2026). Now run by an independent foundation after Steinberger was hired by OpenAI (Feb 2026).

By The Numbers

  • 250K+ GitHub stars — #1 on GitHub, surpassing React and Linux
  • 2.2M weekly npm downloads, 5M+ monthly
  • 1,264+ contributors, 19,199+ commits, 20,000+ forks
  • 2M+ monthly active users, 1.5M+ agents built
  • 142K Discord members
  • 13,000+ skills on ClawHub

Architecture

Hub-and-spoke: Gateway server → Channel adapters (WhatsApp, Telegram, Discord, Slack, iMessage) → Agent Runtime (ReAct loop) → Tool execution (sandboxed). Model-agnostic — supports Claude, GPT, Gemini, Bedrock, DeepSeek, Qwen, Mistral, local models via Ollama.

Key Protocols

  • MCP (Model Context Protocol) — consume any MCP server as a native tool
  • ACP (Agent Client Protocol) — bridge to coding agents (Zed, JetBrains)
  • Skills ecosystem — 13K+ modular extensions on ClawHub

🇺🇸 NVIDIA: The Enterprise Kingmaker

NemoClaw (announced GTC 2026):

  • Open-source enterprise security stack for OpenClaw
  • Single-command deployment of always-on assistants
  • Integrates: OpenShell (sandboxing) + Nemotron (open models) + Red Hat AI
  • Deny-by-default policy — agents can only do what's explicitly allowed

What NVIDIA is really doing: Owning the inference layer of the agent economy. Every OpenClaw agent running NemoClaw = NVIDIA GPUs consuming tokens 24/7. Same playbook as CUDA for training, now for agent inference.
